Introduction

The goal of this assignment is to get to implement 2D rasterization algorithms for lines and polygons. Specifically, you will be asked to implement the Midpoint Line Drawing Algorithm and the parallel triangle fill algorithm as discussed in class.

What you need to do

  • Start editing the assignment template which can be found at: https://editor.p5js.org/RBailey/sketches/zW_0LRpmT. Once the template appears in the p5.org editor, use the File->Duplicate menu item to duplicate the template.  (You will not be able to save any edits until you Duplicate, please be sure that you are logged into your account on p5js.org).

  • Once you are working on the duplicate file, you can edit the file in the browser editor.  You job is to:

    • Implement the Midpoint line drawing Algorithm by filling in code for the drawLine(x1, y1, x2, y2, r, g, b) function.

    • Implement the parallel triangle fill Algorithm by filling in code for the drawTriangle(p1x, p1y, p2x, p2y, p3x, p3y, r, g, b) function.

    • For both functions, you will need to use the setPixel(x, y, r, g, b) function which is supplied, that will set a pixel to a given (r,g,b) value.

  • You are welcome to include new auxiliary functions that your algorithms can call, but please do not edit any of the template code below the line indicated.
  • When you run the sketch, you can press "l" to toggle between the line screen and the triangle screen.  Also, for both screens, hitting the "p" key will toggle between your solution and the Processing solution.  Please keep in mind that the comparison with p5.org is for reference only as we do not know the exact algorithm used by Processing to draw in lines and triangles. so your solution may be a little bit different than the reference solution.  Differences, however, should be minor.
